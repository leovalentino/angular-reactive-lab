Visualizing Angular's Change Detection with Zone.js vs Signals

How it works:

  • Zone.js-based component (left): Uses traditional @Input() binding. Any event in the parent triggers change detection in this child, even if its input hasn't changed.
  • Signals-based component (right): Uses the new input() signal. Only triggers change detection when its specific signal input actually changes.
  • Each component has a checkRender() method that logs to console and flashes the background when called.
